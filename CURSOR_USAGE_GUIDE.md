# 🎯 Cursor AI 使用指南

## 📋 配置方法

### 方式1: 本地仓库配置
在Cursor的文档设置中添加：
```json
{
  "@iso-standards": "d:/01_Git/ISO/**",
  "@automotive-standards": "d:/01_Git/ISO/docs-manifest.json"
}
```

### 方式2: GitHub仓库配置  
```json
{
  "@iso-docs": "https://github.com/your-username/ISO"
}
```
